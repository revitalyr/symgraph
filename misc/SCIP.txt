Соберу для тебя **полный и актуальный список генераторов SCIP** (современная альтернатива LSIF) для **C++**, **Rust** и **скриптовых языков**. Это то, что реально работает сегодня и используется в Sourcegraph.

---

# 🧩 Что такое SCIP‑индексатор
SCIP — бинарный формат семантического индекса кода.  
Он компактнее, быстрее и проще, чем LSIF, и **все новые индексаторы делают именно SCIP**.

---

# 🦀 Rust — генератор SCIP

## ✔️ **scip-rust** (официальный)
Rust‑анализатор умеет экспортировать SCIP напрямую.

### Установка
```bash
cargo install scip-rust
```

### Генерация индекса
```bash
scip-rust index --output dump.scip
```

Работает поверх `rust-analyzer`, использует Cargo metadata, поддерживает workspaces.

---

# 🛠 C++ — генератор SCIP

## ✔️ **scip-clang** (официальный)
Это фронтенд над Clang, который строит SCIP‑индекс.

### Установка
```bash
brew install sourcegraph/scip/scip-clang
```
или собрать из исходников.

### Генерация индекса
```bash
scip-clang index compile_commands.json --output dump.scip
```

Требуется `compile_commands.json` (можно получить через CMake: `-DCMAKE_EXPORT_COMPILE_COMMANDS=ON`).

# 🧪 Скриптовые языки — генераторы SCIP

Ниже — список рабочих индексаторов для динамических языков.

---

## 🐍 Python — **scip-python**
Официальный, активно поддерживается.

### Установка
```bash
pip install scip-python
```

### Генерация
```bash
scip-python index . --output dump.scip
```

---

## 📜 JavaScript / TypeScript — **scip-typescript**
Поддерживает TS и JS (включая `checkJs`).

### Установка
```bash
npm install -g @sourcegraph/scip-typescript
```

### Генерация
```bash
scip-typescript index --project-root . --output dump.scip
```

---

## 🐚 Bash / Shell — **scip-shell**
Поддерживает `.sh`, `.bash`, `.zsh`.

### Установка
```bash
cargo install scip-shell
```

### Генерация
```bash
scip-shell index . > dump.scip
```

---

## 💎 Ruby — **scip-ruby**
Официальный.

### Установка
```bash
gem install scip-ruby
```

### Генерация
```bash
scip-ruby index . --output dump.scip
```

---

## 🐘 PHP — **scip-php**
Официальный.

### Установка
```bash
composer require sourcegraph/scip-php
```

### Генерация
```bash
vendor/bin/scip-php index --output dump.scip
```

---

## 🟨 Lua — **scip-lua** (community)
Работает, но менее зрелый.
